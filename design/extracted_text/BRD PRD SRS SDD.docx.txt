0.5 Ringkasan untuk Pemangku Kepentingan Awam [UMUM]
Portal PORPROV XV Jawa Barat 2026 dirancang sebagai kanal resmi informasi pertandingan, venue, jadwal, LiveScore, klasemen medali, galeri, dan rekomendasi tempat bagi atlet, ofisial, penonton, media, serta masyarakat. Sistem harus mudah dipakai, cepat, aman, ramah perangkat mobile, dan mampu memperbarui skor secara realtime tanpa refresh halaman. Backend admin digunakan untuk mengelola konten, jadwal, venue, bracket, galeri, rekomendasi tempat, dan validasi hasil. Aplikasi mobile koresponden digunakan untuk input skor langsung dari setiap venue dan cabang olahraga.

2. Ringkasan untuk Pemangku Kepentingan [UMUM]
Portal ini harus menjadi pusat informasi resmi PORPROV XV Jawa Barat 2026. Pengunjung dapat melihat jadwal pertandingan, lokasi venue, petunjuk rute, informasi cabang olahraga, maskot Toca dan Toci, LiveScore, klasemen medali, galeri foto-video, serta rekomendasi coffee shop, tempat makan, penginapan, wisata, pusat belanja, dan rumah sakit yang tercantum pada booklet.
Hal-hal Kunci:
1. Realtime: skor dan klasemen tampil otomatis tanpa refresh.
2. Mobile-first: penonton, atlet, media, dan koresponden lebih banyak memakai ponsel.
3. Mudah dikelola: admin dapat memperbarui berita, jadwal, venue, galeri, dan rekomendasi tempat.
4. Akurat: skor harus memiliki status draft, submitted, verified, official, corrected, dan rejected.
5. Aman: akses backend, input skor, dan data pribadi harus dilindungi.
6. SEO kuat: informasi publik mudah ditemukan melalui mesin pencari.

4. BRD — Business Requirements Document
4.1 Latar Belakang
Kota Depok menjadi salah satu tuan rumah PORPROV XV Jawa Barat 2026. Booklet memuat informasi dasar acara, maskot, cabang olahraga, jadwal, venue, peta, dan rekomendasi tempat. Portal digital dibutuhkan agar informasi tersebut dapat diakses cepat, terstruktur, realtime, mudah dicari, dan mudah dikelola oleh panitia.

5.5 User Story dan Acceptance Criteria Gherkin
ID
Judul
Cerita Pengguna
Acceptance Criteria
Prioritas
Tertelusur dari
US-01
Melihat LiveScore
Sebagai pengunjung publik, saya ingin melihat skor pertandingan secara realtime, sehingga saya dapat mengikuti perkembangan pertandingan tanpa refresh halaman.
Sukses: skor tampil maksimal 2 detik setelah event diterima frontend. Validasi: payload skor tidak valid ditolak. Edge case: koneksi putus harus mengambil snapshot terbaru saat tersambung kembali. Keamanan: pengguna publik tidak dapat mengirim skor.
Must
PR-06, BR-02
US-02
Input Skor oleh Koresponden
Sebagai koresponden LiveScore, saya ingin memasukkan skor dari venue melalui aplikasi mobile, sehingga skor dapat dikirim cepat dan dapat diverifikasi.
Sukses: event skor tersimpan sebagai submitted atau verified sesuai aturan. Validasi: match final tidak boleh menerima skor baru tanpa koreksi resmi. Edge case: offline queue mengirim ulang saat koneksi pulih. Keamanan: perangkat belum terdaftar memerlukan verifikasi.
Must
PR-12, BR-03
US-03
Melihat Venue dan Rute
Sebagai penonton, saya ingin melihat venue, peta, rute, jadwal, dan rekomendasi tempat terdekat, sehingga saya dapat merencanakan kunjungan.
Sukses: halaman venue menampilkan peta, alamat, cabor, jadwal, fasilitas, dan tombol rute. Validasi: koordinat belum diverifikasi diberi peringatan. Edge case: Pantai Cimaja diberi label luar wilayah. Keamanan: API key peta dibatasi domain dan kuota.
Must
PR-05, PR-10, BR-04
US-04
Melihat Standings Medali
Sebagai masyarakat, saya ingin melihat peringkat medali per kontingen, sehingga saya mengetahui posisi terbaru secara realtime.
